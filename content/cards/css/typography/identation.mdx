---
question: |
  What are three approaches to indent a paragraph text in CSS?
publishedOn: '2024-07-30T12:00:00+0100'
id: cgrWmm
---

```css
  .text-indent {
    text-indent: 2rem;
  }

  .line-indent::first-line {
    padding-left: 2rem;
  }

  .letter-indent::first-letter {
    padding-left: 2rem;
  }
```
