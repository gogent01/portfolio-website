---
question: |
  How to hyphenate text in CSS?
  What additional property should be used to ensure a text never overflows?
publishedOn: '2024-06-26T12:00:00+0100'
id: tem9aV
---

```css
p {
  hyphens: auto;
  overflow-wrap: break-word; /* Breaks inside the words not processed by the hyphenation algorithm */
}
```
