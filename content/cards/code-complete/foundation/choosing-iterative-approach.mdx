---
question: |
  Which preconditions allow for choosing a Sequential Approach for a project development?
publishedOn: '2024-08-31T12:00:00+0100'
id: Mh1LIA
---
- requirements are poorly understood or are expected to be unstable
- software design is complex or challenging
- dev team is unfamiliar with the application area
- the project has a lot of risk
- long-term predictability is not important
- costs of requirements / architecture / code changes is likely to be low
