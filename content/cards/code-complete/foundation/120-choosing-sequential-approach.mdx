  Which preconditions allow for choosing a Sequential Approach for a project development?
publishedOn: '2024-08-31T12:00:00+0100'
id: SZza6z
---
- requirements are fairly stable
- software design is straightforward and well understood
- dev team is familiar with the application area
- the project has little risk
- long-term predictability is important
- costs of requirements / architecture / code changes is going to be high
