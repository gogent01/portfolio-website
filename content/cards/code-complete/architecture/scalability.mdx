question: |
  Which topics should be covered by a project architecture in the domain of scalability? (Name 4)
publishedOn: '2024-11-10T12:00:00+0100'
id: Iu61Qu
---
The architecture should describe how the system is going to address the growth of:
- number of users
- number of servers
- number of network nodes
- number of database records and transaction volumes

*If the system is not going to be scalable, it should be specified implicitly.*
